Let us define some helper functions.

def laplace(f):
    F = laplace_transform(f, t, s, noconds=True)
    return F

def ilaplace(F):
    f = inverse_laplace_transform(F, s, t)
    return f

def frac_to_tf(frac):
    return TransferFunction(fraction(frac)[0], fraction(frac)[1], s)

System and the control equations#

The diagram above corresponds to a unity-feedback loop with

\[ G(s)=K_p+\frac{K_i}{s}+K_d \]

and plant

\[ H(s)=\frac{1}{s^2 + 2 \zeta \omega_n s +\omega_n^2}. \]

Thus, the closed-loop transfer function is

\[ \boxed{\frac{C(s)}{R(s)} = \frac{K_d s^2+K_p s+K_i}{s^2+\left(K_d+2 \omega_n\right) s^2+\left(K_p+\omega_n^2\right) s+K_i}} \]

After this, we will compute \(c(t)\), \(c'(t)\), \(c(0)\), \(c'(0)\), and \(c(\infty)\) where \(c(t)\) is the inverse Laplace of \(C(s)\).

From \(c(\infty)\), we can get the steady state output of the controlled system while \(c'(0)\) tells us the occurance of derivative kick in the controlled system (non-zero initial velocity).

H = 1 / (s**2+2*zeta*omega_n*s+omega_n**2)
G = Kp + Ki/s + Kd*s        # PID TF
Q = factor(G*H / (1 + G*H)) # CL TF
Q = collect(Q, [s**3, s**2, s])

print("First-order system:")
mprint("H(s)=", latex(H))

print("PID-control:")
mprint("G(s)= ", latex(G))

print("The resulting closed-loop system:")
mprint("Q(s)=", latex(Q))

First-order system:
\[\displaystyle H(s)=\frac{1}{\omega_{n}^{2} + 2 \omega_{n} s \zeta + s^{2}}\]
PID-control:
\[\displaystyle G(s)= K_{d} s + \frac{K_{i}}{s} + K_{p}\]
The resulting closed-loop system:
\[\displaystyle Q(s)=\frac{K_{d} s^{2} + K_{i} + K_{p} s}{K_{i} + s^{3} + s^{2} \left(K_{d} + 2 \omega_{n} \zeta\right) + s \left(K_{p} + \omega_{n}^{2}\right)}\]

At this point, we can tell that the equations are already complicated, thus, finding some equations for further analysis might not be feasible. What we are going to do next is steady state analysis for several combinations of controller terms.

Second-order system with P-control#

Qp = simplify(Q.subs(([Kd, 0],
                      [Ki, 0])))

mprint("Q_P=", latex(Qp))
\[\displaystyle Q_P=\frac{K_{p}}{K_{p} + \omega_{n}^{2} + 2 \omega_{n} s \zeta + s^{2}}\]

For step response, we apply \(1/s\) as its input, perform an inverse Laplace operation to \(C(s)\) and obtain \(c(t)\) as the result. Finally, we compute the steady state-value for the output: \(c_{ss} = \lim_{t \to \infty} c(t)\).

cp = logcombine(ilaplace(Qp * 1/s))
cssp = simplify(limit(cp, t, 'oo'))
c0p  = limit(cp, t, 0)
cp_d  = simplify(diff(cp, t))
c0p_d  = limit(cp_d, t, 0)

mprint("c(t) = ", latex(cp))
mprint("c'(t) = ", latex(cp_d))
mprintb("c_{ss} = ", latex(cssp))
mprintb("c(0) = ", latex(c0p))
mprintb("c'(0) = ", latex(c0p_d))
\[\displaystyle c(t) = K_{p} \left(\frac{\left(- \frac{2 \omega_{n} \zeta}{K_{p} + \omega_{n}^{2}} + \frac{2 K_{p} \omega_{n} \zeta + 2 \omega_{n}^{3} \zeta}{2 \left(K_{p} + \omega_{n}^{2}\right)^{2}}\right) e^{- \frac{t \left(2 K_{p} \omega_{n} \zeta + 2 \omega_{n}^{3} \zeta\right)}{2 \left(K_{p} + \omega_{n}^{2}\right)}} \sin{\left(t \sqrt{K_{p} - \omega_{n}^{2} \zeta^{2} + \omega_{n}^{2}} \right)}}{\sqrt{K_{p} - \omega_{n}^{2} \zeta^{2} + \omega_{n}^{2}}} + \frac{1}{K_{p} + \omega_{n}^{2}} - \frac{e^{- \frac{t \left(2 K_{p} \omega_{n} \zeta + 2 \omega_{n}^{3} \zeta\right)}{2 \left(K_{p} + \omega_{n}^{2}\right)}} \cos{\left(t \sqrt{K_{p} - \omega_{n}^{2} \zeta^{2} + \omega_{n}^{2}} \right)}}{K_{p} + \omega_{n}^{2}}\right)\]
\[\displaystyle c'(t) = \frac{K_{p} e^{- \omega_{n} t \zeta} \sin{\left(t \sqrt{K_{p} - \omega_{n}^{2} \zeta^{2} + \omega_{n}^{2}} \right)}}{\sqrt{K_{p} - \omega_{n}^{2} \zeta^{2} + \omega_{n}^{2}}}\]
\[\displaystyle \boxed{c_{ss} = \frac{K_{p}}{K_{p} + \omega_{n}^{2}}}\]
\[\displaystyle \boxed{c(0) = 0}\]
\[\displaystyle \boxed{c'(0) = 0}\]

Our conclusions:

  • no derivative kick

  • smooth start

  • zero initial velocity

Next, we can write the steady state error as: \(e_{ss} = 1 - c_{ss}\)

mprintb("e_{ss}=", latex(simplify(1-cssp)))
\[\displaystyle \boxed{e_{ss}=\frac{\omega_{n}^{2}}{K_{p} + \omega_{n}^{2}}}\]

Second-order system with PD-control#

Closed-loop transfer function:

Qpd = simplify(Q.subs(Ki, 0))
mprint("Q_{PD}=", latex(Qpd))
\[\displaystyle Q_{PD}=\frac{K_{d} s + K_{p}}{K_{p} + \omega_{n}^{2} + s^{2} + s \left(K_{d} + 2 \omega_{n} \zeta\right)}\]

Next, we perform an inverse Laplace operation to \(C(s)\) and obtain \(c(t)\) as the result. Additionally, we will also compute the steady state-value for the output (\(c_{ss}\)).

A phenomenon that can be observed in a derivative control is the “kick” that happens when a step input is applied to the controlled system. Because of the kick, system output does not start from zero.

cpd    = logcombine(ilaplace(Qpd * 1/s))
cpd_d  = powsimp(factor((diff(cpd, t))))

csspd = limit(cpd, t, 'oo')
c0pd  = limit(cpd, t, 0)
c0pd_d  = limit(cpd_d, t, 0)

mprint("\\small c(t) = ", latex(cpd))

mprintb("c(0) = ", latex(c0pd))
mprintb("c'(0) = ", latex(c0pd_d))
\[\displaystyle \small c(t) = \frac{K_{p}}{K_{p} + \omega_{n}^{2}} - \frac{K_{p} e^{- \frac{t \left(K_{d} K_{p} + K_{d} \omega_{n}^{2} + 2 K_{p} \omega_{n} \zeta + 2 \omega_{n}^{3} \zeta\right)}{2 \left(K_{p} + \omega_{n}^{2}\right)}} \cos{\left(\frac{t \sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}}}{2} \right)}}{K_{p} + \omega_{n}^{2}} + \frac{2 \left(\frac{K_{p} \left(K_{d} K_{p} + K_{d} \omega_{n}^{2} + 2 K_{p} \omega_{n} \zeta + 2 \omega_{n}^{3} \zeta\right)}{2 \left(K_{p} + \omega_{n}^{2}\right)^{2}} + \frac{K_{d} \omega_{n}^{2} - 2 K_{p} \omega_{n} \zeta}{K_{p} + \omega_{n}^{2}}\right) e^{- \frac{t \left(K_{d} K_{p} + K_{d} \omega_{n}^{2} + 2 K_{p} \omega_{n} \zeta + 2 \omega_{n}^{3} \zeta\right)}{2 \left(K_{p} + \omega_{n}^{2}\right)}} \sin{\left(\frac{t \sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}}}{2} \right)}}{\sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}}}\]
\[\displaystyle \small c'(t) = \frac{\left(- K_{d}^{2} \sin{\left(\frac{t \sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}}}{2} \right)} - 2 K_{d} \omega_{n} \zeta \sin{\left(\frac{t \sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}}}{2} \right)} + K_{d} \sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}} \cos{\left(\frac{t \sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}}}{2} \right)} + 2 K_{p} \sin{\left(\frac{t \sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}}}{2} \right)}\right) e^{- \frac{t \left(\frac{K_{d} K_{p}}{2} + \frac{K_{d} \omega_{n}^{2}}{2} + K_{p} \omega_{n} \zeta + \omega_{n}^{3} \zeta\right)}{K_{p} + \omega_{n}^{2}}}}{\sqrt{- K_{d}^{2} - 4 K_{d} \omega_{n} \zeta + 4 K_{p} - 4 \omega_{n}^{2} \zeta^{2} + 4 \omega_{n}^{2}}}\]
\[\displaystyle \boxed{c_{ss} = \frac{K_{p}}{K_{p} + \omega_{n}^{2}}}\]

Second-order system with PI-control#

Let us write down the closed-loop transfer function of a PI-controlled second-order system:

Qpi = simplify(Q.subs(Kd, 0))
mprintb("Q_{PI} =", latex(Qpi))
\[\displaystyle \boxed{Q_{PI} =\frac{K_{i} + K_{p} s}{K_{i} + 2 \omega_{n} s^{2} \zeta + s^{3} + s \left(K_{p} + \omega_{n}^{2}\right)}}\]

At this point we can no longer afford performing Inverse Laplace operation with SymPy since it is now involving finding root of a third orde polynomials. In order to find the steady state ouput, we will use final value theorem.

Cpi = Qpi*1/s
csspi = limit(s * Cpi, s, 0) 
mprint("C(s) =", latex(Cpi))
mprintb("c_{ss} = c(\\infty) =", latex(csspi))
\[\displaystyle C(s) =\frac{K_{i} + K_{p} s}{s \left(K_{i} + 2 \omega_{n} s^{2} \zeta + s^{3} + s \left(K_{p} + \omega_{n}^{2}\right)\right)}\]
\[\displaystyle \boxed{c_{ss} = c(\infty) =1}\]

At steady state, the output is 1 which means PI controlled system does not have steady state error.

Next, we wll introduce \(-a_1\), \(-a_2\), and \(-a_3\) as the roots of: \(\left(K_i+2 \omega_n s^2 \zeta+s^3+s\left(K_p+\omega_n^2\right)\right)\)

\[C(s)=\frac{K_i+K_p s}{s\left(K_i+2 \omega_n s^2 \zeta+s^3+s\left(K_p+\omega_n^2\right)\right)} = \frac{K_i+K_p s}{s(s+a_1)(s+a_3)(s+a_2)} \]
a1, a2, a3 = symbols('a1 a2 a3', complex=True)
Cpi_fac = (Kp*s + Ki) / (s*(s + a1)*(s + a2)*(s + a3))

mprint("C(s) = ", latex(Cpi_fac))

Cpi_pf = apart(Cpi_fac, s)

print("\nPartial fraction form")
mprint("C(s) = ", latex(Cpi_pf))

cpi = logcombine(inverse_laplace_transform(Cpi_pf, s, t))

print("\nInverse Laplace")
mprintb("c(t) = ", latex(cpi))
\[\displaystyle C(s) = \frac{K_{i} + K_{p} s}{s \left(a_{1} + s\right) \left(a_{2} + s\right) \left(a_{3} + s\right)}\]
Partial fraction form
\[\displaystyle C(s) = \frac{K_{i}}{a_{1} a_{2} a_{3} s} + \frac{- K_{i} + K_{p} a_{3}}{a_{3} \left(a_{1} - a_{3}\right) \left(a_{2} - a_{3}\right) \left(a_{3} + s\right)} - \frac{- K_{i} + K_{p} a_{2}}{a_{2} \left(a_{1} - a_{2}\right) \left(a_{2} - a_{3}\right) \left(a_{2} + s\right)} + \frac{- K_{i} + K_{p} a_{1}}{a_{1} \left(a_{1} - a_{2}\right) \left(a_{1} - a_{3}\right) \left(a_{1} + s\right)}\]
Inverse Laplace
\[\displaystyle \boxed{c(t) = \frac{K_{i}}{a_{1} a_{2} a_{3}} + \frac{\left(- K_{i} + K_{p} a_{3}\right) e^{- a_{3} t}}{a_{3} \left(a_{1} - a_{3}\right) \left(a_{2} - a_{3}\right)} - \frac{\left(- K_{i} + K_{p} a_{2}\right) e^{- a_{2} t}}{a_{2} \left(a_{1} - a_{2}\right) \left(a_{2} - a_{3}\right)} + \frac{\left(- K_{i} + K_{p} a_{1}\right) e^{- a_{1} t}}{a_{1} \left(a_{1} - a_{2}\right) \left(a_{1} - a_{3}\right)}}\]

We can now compute the intial output \(c(0)\).

c0pi = simplify(cpi.subs(t, 0))
cpi_d = simplify(diff(cpi, t))
c0pi_d = limit(cpi_d, t, 0)

mprintb("c(0) = ", latex(c0pi))
mprint("c'(t) = ", latex(cpi_d))
mprintb("c'(0) = ", latex(c0pi_d))
\[\displaystyle \boxed{c(0) = 0}\]
\[\displaystyle c'(t) = \frac{\left(\left(- K_{i} + K_{p} a_{2}\right) \left(a_{1} - a_{3}\right) e^{t \left(a_{1} + a_{3}\right)} + \left(K_{i} - K_{p} a_{1}\right) \left(a_{2} - a_{3}\right) e^{t \left(a_{2} + a_{3}\right)} + \left(K_{i} - K_{p} a_{3}\right) \left(a_{1} - a_{2}\right) e^{t \left(a_{1} + a_{2}\right)}\right) e^{- t \left(a_{1} + a_{2} + a_{3}\right)}}{\left(a_{1} - a_{2}\right) \left(a_{1} - a_{3}\right) \left(a_{2} - a_{3}\right)}\]
\[\displaystyle \boxed{c'(0) = 0}\]

Our conclusions:

  • no derivative kick

  • smooth start

  • zero initial velocity

Next, we can write the steady state error as: \(e_{ss} = 1 - c_{ss}\)

In this case, we will need to first find \(a_1\), \(a_2\), and \(a_3\) (perhaps numerically). Then, we plug them into the later equation to find \(c(t)\). Let us take some arbitrary numerical values so we can plot the controlled system’s reponses.

mprintb("c(0) = ", latex(c0pid))
mprint("c'(t) = ", latex(cpid_d))
mprintb("c'(0) = ", latex(c0pid_d))
\[\displaystyle \boxed{c(0) = 0}\]
\[\displaystyle c'(t) = \frac{\left(\left(a_{1} - a_{2}\right) \left(K_{d} a_{3}^{2} + K_{i} - K_{p} a_{3}\right) e^{t \left(a_{1} + a_{2}\right)} + \left(a_{1} - a_{3}\right) \left(- K_{d} a_{2}^{2} - K_{i} + K_{p} a_{2}\right) e^{t \left(a_{1} + a_{3}\right)} + \left(a_{2} - a_{3}\right) \left(K_{d} a_{1}^{2} + K_{i} - K_{p} a_{1}\right) e^{t \left(a_{2} + a_{3}\right)}\right) e^{- t \left(a_{1} + a_{2} + a_{3}\right)}}{\left(a_{1} - a_{2}\right) \left(a_{1} - a_{3}\right) \left(a_{2} - a_{3}\right)}\]
\[\displaystyle \boxed{c'(0) = K_{d}}\]

Our conclusions:

  • derivative kick

  • non-smooth start

  • non-zero initial velocity

Next, we can write the steady state error as: \(e_{ss} = 1 - c_{ss}\)

mprintb("e_{ss}=", latex(simplify(1-csspid)))
\[\displaystyle \boxed{e_{ss}=0}\]

Interactive PID Response of a Second-Order Plant#

This sectoin demonstrates the step response of a second-order plant controlled using a PID controller. The goal is to observe how the controller gains \(K_p\), \(K_i\), and \(K_d\) affect the closed-loop response.

The plant is modeled as a standard second-order system with these parameters:

  • \(\zeta\) is the damping ratio,

  • \(\omega_n\) is the natural frequency.

The notebook then plots the response \(c(t)\) to a unit-step input.

The notebook also calculates common time-domain response characteristics, including:

  • rise time,

  • peak time,

  • maximum overshoot,

  • settling time using the 5% band,

  • settling time using the 2% band,

  • closed-loop pole locations.

import numpy as np
import matplotlib.pyplot as plt
import ipywidgets as widgets
from IPython.display import display
from scipy import signal
from scipy.optimize import brentq


def closed_loop_tf(Kp, Ki, Kd, zeta, wn):
    """Return the closed-loop transfer function C(s)/R(s)."""

    # Numerator:
    # Kd*s^2 + Kp*s + Ki
    num = [Kd, Kp, Ki]

    # Denominator:
    # s^3 + (Kd + 2*zeta*wn)*s^2 + (Kp + wn^2)*s + Ki
    den = [
        1.0,
        Kd + 2.0*zeta*wn,
        Kp + wn**2,
        Ki
    ]

    sys = signal.TransferFunction(num, den)
    return sys, num, den


def first_crossing_time(t, y, level):
    """Find first time where y(t) crosses a specified level."""
    f = y - level

    for k in range(len(t) - 1):
        if f[k] == 0:
            return t[k]

        if f[k] * f[k + 1] < 0:
            t_left = t[k]
            t_right = t[k + 1]

            return brentq(
                lambda tau: np.interp(tau, t, y) - level,
                t_left,
                t_right
            )

    return None


def settling_time(t, y, y_final, tol):
    """Settling time: first time after which response stays inside ±tol band."""
    lower = (1.0 - tol) * y_final
    upper = (1.0 + tol) * y_final

    inside = (y >= lower) & (y <= upper)

    for k in range(len(t)):
        if np.all(inside[k:]):
            return t[k]

    return None


def plot_pid_response(Kp=6.0, Ki=3.0, Kd=1.0,
                      zeta=0.4, wn=5.0,
                      t_final=10.0):

    # Basic validation
    if wn <= 0:
        print("Please use wn > 0.")
        return

    if zeta <= 0:
        print("Please use zeta > 0.")
        return

    if Ki <= 0:
        print("Please use Ki > 0 for integral action and unit steady-state tracking.")
        return

    if t_final <= 0:
        print("Please use t final > 0.")
        return

    sys, num, den = closed_loop_tf(Kp, Ki, Kd, zeta, wn)

    # Closed-loop poles
    poles = np.roots(den)

    # Stability check
    stable = np.all(np.real(poles) < 0)

    # Time response
    t = np.linspace(0, t_final, 2000)
    t, c = signal.step(sys, T=t)

    # For this PID closed-loop with Ki > 0 and stable poles:
    c_inf = 1.0

    # Rise time: first time c(t) reaches the unit reference
    tr = first_crossing_time(t, c, 1.0)

    # First crossing of 95% and 98%
    t95 = first_crossing_time(t, c, 0.95*c_inf)
    t98 = first_crossing_time(t, c, 0.98*c_inf)

    # Settling time using 5% and 2% bands
    Ts_5 = settling_time(t, c, c_inf, 0.05)
    Ts_2 = settling_time(t, c, c_inf, 0.02)

    # Peak information
    peak_index = np.argmax(c)
    t_peak = t[peak_index]
    c_peak = c[peak_index]
    Mp = c_peak - c_inf
    percent_overshoot = 100.0 * Mp / c_inf

    # Plot
    plt.figure(figsize=(9, 5))
    plt.plot(t, c, linewidth=2, label=r"$c(t)$")

    plt.axhline(1.0, linestyle="--", linewidth=1, label="unit reference")

    # Settling bands
    plt.axhline(1.05, linestyle="--", linewidth=1, label=r"$\pm 5\%$ band")
    plt.axhline(0.95, linestyle="--", linewidth=1)

    plt.axhline(1.02, linestyle="-.", linewidth=1, label=r"$\pm 2\%$ band")
    plt.axhline(0.98, linestyle="-.", linewidth=1)

    # Rise time marker
    if tr is not None:
        plt.axvline(tr, linestyle="--", linewidth=1,
                    label=rf"$t_r={tr:.3g}$")
        plt.plot(tr, 1.0, "o")

    # 95% and 98% first-crossing markers
    if t95 is not None:
        plt.plot(t95, 0.95, "o", label=rf"$t_{{95}}={t95:.3g}$")

    if t98 is not None:
        plt.plot(t98, 0.98, "o", label=rf"$t_{{98}}={t98:.3g}$")

    # Peak marker
    plt.plot(t_peak, c_peak, "s", label=rf"peak at $t_p={t_peak:.3g}$")

    # Settling time markers
    if Ts_5 is not None:
        plt.axvline(Ts_5, linestyle="--", linewidth=1,
                    label=rf"$T_{{s,5\%}}={Ts_5:.3g}$")

    if Ts_2 is not None:
        plt.axvline(Ts_2, linestyle="-.", linewidth=1,
                    label=rf"$T_{{s,2\%}}={Ts_2:.3g}$")

    plt.xlabel("Time, t")
    plt.ylabel("Response, c(t)")
    plt.title("PID-Controlled Second-Order Plant")
    plt.grid(True)
    plt.legend(loc="best")
    plt.show()

    # Print information
    print("Closed-loop transfer function:")
    print("Numerator coefficients:")
    print(f"  {num}")
    print("Denominator coefficients:")
    print(f"  {den}")
    print()

    print("Closed-loop poles:")
    for k, p in enumerate(poles, start=1):
        print(f"p{k} = {p:.6g}")
    print()

    print(f"Stable?      = {stable}")
    print(f"zeta         = {zeta:.6g}")
    print(f"wn           = {wn:.6g}")
    print()

    print(f"c(0)         = {c[0]:.6g}")
    print(f"c(infty)    = {c_inf:.6g}")
    print(f"max c(t)    = {c_peak:.6g}")
    print(f"t_peak      = {t_peak:.6g}")
    print(f"Mp          = {Mp:.6g}")
    print(f"%OS         = {percent_overshoot:.6g} %")
    print()

    if tr is not None:
        print(f"rise time tr, c(t)=1        = {tr:.6g}")
    else:
        print("rise time tr, c(t)=1        = not found in plotted time range")

    if t95 is not None:
        print(f"t95 first crossing          = {t95:.6g}")
    else:
        print("t95 first crossing          = not found in plotted time range")

    if t98 is not None:
        print(f"t98 first crossing          = {t98:.6g}")
    else:
        print("t98 first crossing          = not found in plotted time range")

    if Ts_5 is not None:
        print(f"settling time 5%            = {Ts_5:.6g}")
    else:
        print("settling time 5%            = not settled in plotted time range")

    if Ts_2 is not None:
        print(f"settling time 2%            = {Ts_2:.6g}")
    else:
        print("settling time 2%            = not settled in plotted time range")
